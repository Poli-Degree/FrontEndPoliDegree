POLI DEGREE

Bienvenidos

Poli Degree es una aplicación web desarrollada especialmente para los proyectos de grado de los estudiantes del Politecnico Grancolombiano enfocada a los que desean escoger esta opción para graduarse. La función principal de esta aplicación web es de llevar un seguimiento y proceso del proyecto del estudiante registrado en la aplicación, además de que el estudiante gestionara su proyecto los docentes y evaluadores se encargaran de asignar las tareas, calificaciones y observaciones que tendrá cada proyecto.

Recuerda que en PoliDegree podrás:

  • Crear y alojar proyectos de grado de estudiantes para su continuo desarrollo.
  • Optimizar el proceso de inscripciones a proyectos de grado.
  • Mejorar la experiencia a la hora de llevar a cabo un proyecto de grado.
  • Tener una experiencia de usuario que sea cómoda y fácil de usar.
  • Mejorar el sistema de tareas y compromisos por cumplir para mejorar proyectos.
  • Poder visualizar calificaciones y comentarios de docentes y evaluadores.
